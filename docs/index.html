Anthropic社の生成AI

会話でいろいろできる

どこでも使える

Web / スマホ / PC で使えて、用途に合わせて気軽に始められます。

Claudeの2つのエージェント

Cowork

誰でも使いやすい。チャット感覚でPC操作・ブラウザ操作・ファイル作業まで進められます。

Claude Code

エンジニア向け。ターミナルで動作し、コード編集や開発作業を強力に進められます。

初心者はまず Cowork を触ってみよう!

「相談」から「依頼」へ

ChatGPT

  • 優秀な「相談役」
  • アイデアや文章を出す
  • 最後は人が手を動かす

Claude Cowork

  • 頼れる「代行者」
  • PCやブラウザも扱える
  • AIが実務を前に進める

すごいぞ！Claude Cowork

できることは大きく4つ。ここを押さえると全体像がつかめます。

ファイルを扱える

パソコンの中の資料やデータを読んで、整理・比較・要約まで進められます。

ブラウザを扱える

検索やページ確認をしながら、情報収集から整理までつなげられます。

スキルで自動化

一度教えた手順を覚えて、同じ作業を一言で呼び出せます。

スケジュール実行

定期タスクを時間指定で回して、毎日の面倒を減らせます。

パソコンの中のファイルを操作できる

ファイルの整理
デスクトップやダウンロードファイル、散らかっていませんか？ Claudeに頼んで整理してみましょう。
ファイルを読み取って資料の作成
データは集めたけど、分析とか資料作成ってめんどくさいですよね。Claudeに任せちゃおう。

ブラウザを扱える

ログイン状態を保ったままブラウザを操作できる
普段使っているブラウザのログイン状態を引き継いだまま、ページ確認や情報取得を進められます。
使い方は無限大
学長高配当マガジンから最新の銘柄を取得する、楽天カードの使用状況を見て脂質管理表に科目ごとに記入してもらう、などができます。
ただし制限されている操作もある
株の実際の売買、Amazonでの買い物の決定など、重要な最終判断や購入系の操作は制限されています。

Claude in Chrome 拡張機能: Chrome Web Store で開く

スキル機能で自動化できる

具体例: さっきの高配当株チェックを覚えさせる
学長高配当株マガジンを見て、最新銘柄をヤフーファイナンスで確認する流れをそのまま覚えさせられます。
次回から一言で呼べる
次からは「高配当株チェックして」だけで、同じ流れを再現しやすくなります。
毎回の作業時間を大幅に圧縮できる
毎回同じサイトを見る作業、同じ観点で確認する作業を省力化できます。

スケジュールで定期タスクを回せる

具体例: 毎週火曜日に高配当株チェック
毎週火曜日に、学長高配当株マガジンの最新銘柄を確認する定期タスクとして回せます。
抜け漏れを減らせる
毎週見たい情報、毎月やる確認を「思い出したらやる」から卒業できます。
小さな面倒を手放せる
習慣にしたいチェックほど、自動化するとラクさを実感しやすいです。
