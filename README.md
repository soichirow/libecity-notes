# Claude Cowork Slide Deck

リベシティ向けの Claude / Claude Cowork 紹介スライドです。
静的な HTML / CSS / JavaScript で構成されており、`docs/` 配下をそのまま GitHub Pages などで公開できます。

## Files

- `docs/index.html`: スライド本体
- `docs/style.css`: スライド全体のスタイル
- `docs/script.js`: スライド送り、コピー機能、タイマー制御
- `docs/*.png`, `docs/favicon.svg`: 画像アセット

## Local Preview

`docs/index.html` をブラウザで直接開くと確認できます。

## Notes

- スライドはキーボードの左右キー、スペースキー、タッチ操作で移動できます
- 右下のナビゲーションからページ送りとタイマー操作ができます
- `docs/` を公開対象にすると GitHub Pages と相性がよい構成です
